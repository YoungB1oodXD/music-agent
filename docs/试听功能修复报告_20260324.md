## 一、问题诊断

### 1.1 初始问题

1. **首次推荐通常不能试听** - 只有显式说"能试听"才出现可试听标签
2. **点击播放没有声音** - UI 显示播放但实际无声
3. **多首同时播放** - 播放控制逻辑错误
4. **播放失败但 UI 状态不正确** - 状态基于推测而非真实事件

### 1.2 根因分析

| 问题 | 根因 |
|------|------|
| 首次推荐不能试听 | `hybrid_recommend_tool` 丢失 `is_playable`/`audio_url` 字段 |
| 点击没有声音 | `audio_mapping.json` 路径前缀错误 + Vite proxy 未配置 `/audio` |
| 多首同时播放 | 每个卡片独立管理 audio 元素，无全局协调 |
| UI 状态错误 | 状态基于推测，未监听真实 audio 事件 |

## 二、修复方案

### 2.1 后端修复

#### 文件: `src/tools/semantic_search_tool.py`

**修改内容**:
1. 添加 `_AUDIO_STATIC_ROOT` 常量，明确静态文件根目录
2. 添加 `_MIN_AUDIO_FILE_SIZE = 1000`，最小文件大小阈值
3. 重写 `_get_audio_info()` 函数，验证文件存在性和大小

**关键代码**:
```python
_AUDIO_STATIC_ROOT = Path(__file__).parent.parent.parent / "dataset" / "raw" / "fma_small"
_MIN_AUDIO_FILE_SIZE = 1000

def _get_audio_info(track_id: str) -> dict[str, object]:
    mapping = _get_audio_mapping()
    track_id_str = str(track_id)
    if track_id_str not in mapping:
        return {"is_playable": False, "audio_url": None}
    
    raw_path = mapping[track_id_str]
    if raw_path.startswith("fma_small/"):
        audio_path = raw_path[len("fma_small/"):]
    else:
        audio_path = raw_path
    
    # 验证文件真实存在
    full_file_path = _AUDIO_STATIC_ROOT / audio_path
    if not full_file_path.exists():
        return {"is_playable": False, "audio_url": None}
    
    file_size = full_file_path.stat().st_size
    if file_size < _MIN_AUDIO_FILE_SIZE:
        return {"is_playable": False, "audio_url": None}
    
    return {"is_playable": True, "audio_url": f"/audio/{audio_path}"}
```

**修改内容**:
1. `HybridItem` TypedDict 增加 `is_playable` 和 `audio_url` 字段
2. 合并语义结果时传递这些字段

### 2.2 前端修复

#### 文件: `frontend/src/contexts/AudioPlayerContext.tsx` (新建)

**目的**: 实现全局单例音频播放器，确保同时只能播放一首歌曲

**关键代码**:
```tsx
export function AudioPlayerProvider({ children }: { children: React.ReactNode }) {
  const [playingTrackId, setPlayingTrackId] = useState<string | null>(null);
  const audioRef = useRef<HTMLAudioElement | null>(null);
  const pendingTrackIdRef = useRef<string | null>(null);

  useEffect(() => {
    const audio = new Audio();
    audio.preload = 'none';
    
    // 真实事件驱动状态同步
    audio.addEventListener('play', () => {
      if (pendingTrackIdRef.current) {
        setPlayingTrackId(pendingTrackIdRef.current);
        pendingTrackIdRef.current = null;
      }
    });
    audio.addEventListener('pause', () => setPlayingTrackId(null));
    audio.addEventListener('ended', () => {
      setPlayingTrackId(null);
      pendingTrackIdRef.current = null;
    });
    audio.addEventListener('error', () => {
      console.error(`Audio error (code ${audio.error?.code})`);
      setPlayingTrackId(null);
      pendingTrackIdRef.current = null;
    });
    
    audioRef.current = audio;
    return () => { audio.pause(); audio.src = ''; };
  }, []);

  const playTrack = useCallback(async (trackId: string, audioUrl: string) => {
    const audio = audioRef.current;
    if (!audio) return { success: false, error: 'Player not initialized' };

    // 点击同一首则暂停
    if (playingTrackId === trackId) {
      audio.pause();
      return { success: true };
    }

    // 暂停当前，播放新的
    audio.pause();
    audio.src = audioUrl;
    pendingTrackIdRef.current = trackId;
    
    try {
      await audio.play();
      return { success: true };
    } catch (error) {
      pendingTrackIdRef.current = null;
      return { success: false, error: String(error) };
    }
  }, [playingTrackId]);

  return (
    <AudioPlayerContext.Provider value={{ playingTrackId, playTrack, pauseTrack, isPlaying }}>
      {children}
    </AudioPlayerContext.Provider>
  );
}
```
